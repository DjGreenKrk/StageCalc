# GreenCrew Tools Documentation Pack

Pakiet dokumentów bazowych dla aplikacji GreenCrew Tools.

## Struktura

```text
branding/
  GREENCREW_BRANDING.md
  DESIGN_SYSTEM.md
  ICONOGRAPHY.md
  WRITING_GUIDELINES.md

development/
  ARCHITECTURE.md
  FLUTTER_STANDARDS.md
  CODING_STANDARDS.md

project/
  BRANDING_STAGECALC.md
```

## Sposób użycia

Do każdego nowego projektu warto wrzucić:

- `branding/GREENCREW_BRANDING.md`
- `branding/DESIGN_SYSTEM.md`
- `branding/ICONOGRAPHY.md`
- `branding/WRITING_GUIDELINES.md`
- `development/ARCHITECTURE.md`
- `development/FLUTTER_STANDARDS.md`
- `development/CODING_STANDARDS.md`
- własny plik `BRANDING.md` dla konkretnej aplikacji.

Plik `project/BRANDING_STAGECALC.md` jest gotowym brandingiem aplikacji StageCalc.
